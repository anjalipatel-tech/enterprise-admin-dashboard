# 🚀 Enterprise Admin Dashboard

A modern, responsive, and feature-rich **Enterprise Admin Dashboard** built with **React.js**, **Vite**, and **Bootstrap 5**. The dashboard provides a clean user interface for managing business analytics, authentication, user information, and interactive components.

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/anjalipatel-tech/enterprise-admin-dashboard.git
```

### Navigate to Project

```bash
cd enterprise-admin-dashboard
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
